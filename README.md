# 助词

一个简单的宋词填词工具，[点击进入](https://chienmy.github.io/SongCiApp/)。

## 功能特点

- 收录大多数词牌和作者体例
- 自动格律检查，四种韵书可选
- 不够智能但有用的组词提示

大概能满足一般的填词需求。

## 重要提示

- 平仄检查：多音字请自行判断
- 韵部检查：未考虑邻韵通押情形
- 数据保存：关闭网页前务必自行保存作品
